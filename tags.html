Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
header header inclode a header
div division Group block of elements in a document used for styling purposes
ul un-ordered list create a bulleted list
ol ordered list create a numbered list
li list item create a list item

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make a strong text <strong>Hello, World!</strong> Hello, World!
a anchor make a hyperlink <a href="https://www.learnenough.com/html-tutorial">Link to learn enough HTML to be Dangerous!</a> Link to learn enough HTML to be Dangerous!
img Image Include an image <img src="images/small_twitter_logo.png">
code code Display text with the same width <code>Testing</code> Testing
span span Group inline elements in a document used for styling purposes <span>Testing</span> Testing
